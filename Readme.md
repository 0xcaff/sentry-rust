Rust Sentry Lib
==========

[Sentry Service](https://www.getsentry.com/) now available for rust ;)


## Usage

Add this to your `Cargo.toml`:

```toml
[dependencies]
sentry = "0.1.0"
```

and this to your crate root:

```rust
extern crate sentry;
```
